# Google Store Clone

A pixel-perfect responsive clone of the Google Store landing page built with HTML and CSS.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Variables, Media Queries)

## What I Learned

- CSS Grid and Flexbox for complex layouts
- CSS Variables for consistent theming
- Media Queries for responsive design (992px, 768px, 480px breakpoints)
- Position sticky for fixed navbar
- CSS-only hamburger menu using checkbox hack
- Pseudo-classes (:hover, :nth-child) for styling
